# SoundsLoud

### Technologies
#### Front End
- JavaScript
- React-Redux
- CSS
- [ShapeDivider App](https://www.shapedivider.app/) (for splash page divider)
- Heroku (Hosting)

#### Back End
- Express.js
- PostgreSQL (Database)
- Sequelize.js
- Express Validator Library
- CSURF Library
